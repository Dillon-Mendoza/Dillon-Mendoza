## What I'm Building

**[homelab](https://github.com/Dillon-Mendoza/homelab)** —
A six-device environment running Fedora, Ubuntu, and Raspberry Pi
OS across a Tailscale mesh network. Hardened with FirewallD and
UFW, SELinux enforcing, default-deny ACL with purpose-built device
tags and a tiered trust hierarchy. Pi-hole running as a dedicated
DNS filtering node for the mesh.

**[mudd-labs](https://github.com/Dillon-Mendoza/mudd-labs)** —
The build environment for tools I actually use and deploy across
the mesh. Three active projects:

- **ClearMudd** — SSH audit log parser and access intelligence tool.
  Parses journalctl output, surfaces failed attempts, accepted keys,
  and session patterns. Brute force detection and Discord notification
  in progress.
- **Muddroom** — Django dashboard serving live device reachability and
  service status across the mesh. Port checks via webhook, Netdata
  integration, served over Tailscale with HTTPS.
- **mudd** — CLI session journaling tool. Project-scoped,
  terminal-native. Logs what I built, what broke, and what I learned —
  tied to the working directory. Specced, build not started.
